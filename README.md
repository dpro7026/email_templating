## Create an example express app
'''sudo npm install -g express-generator'''

'''express -c stylus express_example'''

'''cd express_example && npm install'''

'''npm start'''

or for server restart upon saving:
'''nodemon app.js'''

View on:
http://127.0.0.1:3000/
